# Api Gateway

An API gateway build it node.js calling two api's fake.

## Installation

Use the npm to install the packages in three files.

```bash
cd ./gateway

npm install
```
To install the gateway.

```bash
cd ./api.users

npm install
```
To install the user API fake.

```bash
cd ./api.products

npm install
```
To install the products API fake.
